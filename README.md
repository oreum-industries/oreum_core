## Oreum Core Tools `oreum_core`

This is an ever-growing package of core tools for use on client projects by
Oreum Industries.

### Notes

+ Uses a scientific Python stack for scripting
+ Hosted on

+ See CONTRIBUTORS.md for list of contributors
